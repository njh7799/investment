# QQQ ZigZag 저점 확정 종료

이 추가 규칙은 [시장 상태 규칙](../market-regimes.md)에 공황의 장 마감 종료 조건 하나를 추가한다. 달력 만료와 QQQ 8거래일 연속 상승 조건은 그대로 유지한다.

대공황에는 이 추가 규칙을 적용하지 않는다. 대공황은 QQQ ZigZag 저점이 확정되어도 종료하지 않으며, 기본 시장 상태 규칙의 달력 만료 또는 QQQ 8거래일 연속 상승 조건으로만 종료한다.

## 종료 조건

최근 IXIC 급락 발생일부터 QQQ 조정 종가의 저점 후보를 추적한다. 더 낮은 조정 종가가 나오면 그날의 종가와 날짜로 저점 후보를 갱신한다.

이후 QQQ 조정 종가의 상승 폭이 다음 두 조건을 모두 만족하면 저점 후보를 ZigZag 저점으로 확정하고, 그날 장 마감 후 공황을 종료한다.

```text
QQQ 현재 종가 - QQQ 저점 후보 종가 >= 확인일 QQQ ATR(14) × 4
AND
QQQ 현재 종가 / QQQ 저점 후보 종가 - 1 >= 10%
```

- QQQ ATR(14)은 [README](../../README.md)의 정의와 동일하게 조정 High, Low, Close의 True Range 14개 단순이동평균으로 계산한다.
- ATR 배수에는 반전 확인일의 QQQ ATR(14)을 사용한다.
- 두 조건은 경계값과 정확히 같은 경우도 충족한 것으로 본다.
- ATR(14)에 필요한 14개 값이 쌓이기 전에는 이 종료 신호를 만들지 않는다.
- 공황 중 IXIC 급락이 다시 발생하면 저점 후보를 해당 급락일의 QQQ 조정 종가로 초기화한다.
- 공황이 대공황으로 전환되면 QQQ 저점 후보 추적과 ZigZag 종료 판정을 중단한다.

같은 장 마감에 IXIC 급락과 QQQ ZigZag 저점 확정이 함께 성립하면 급락을 우선한다. 기존 상태를 종료하지 않고 새 급락일을 기준으로 상태와 종료 조건을 갱신한다.

이 규칙으로 시장 상태가 종료된 이후의 목표 비중과 주문은 기본 투자 규칙을 따른다. [시장 상태 종료 후 전액 투자](full-investment-after-regime.md)를 함께 사용하면 종료 판정에는 이 문서를, 종료 후 복귀 방식에는 해당 추가 규칙을 적용한다.
