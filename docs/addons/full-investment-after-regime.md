# 시장 상태 종료 후 전액 투자

이 추가 규칙은 [기본 투자 규칙](../investment-rules.md)의 공황·대공황 종료 후 복귀 방식만 변경한다. 나머지 시장 상태 판정, 가격 구간, 목표 비중과 체결 규칙은 기본 규칙을 따른다.

## 종료 직후

- 공황 또는 대공황 마지막 거래일의 다음 거래일 시가에 TQQQ 100%, 현금 0%로 즉시 리밸런싱한다.
- 이 주문에는 종가·시가 방향 재확인 조건을 적용하지 않는다.
- 해당 거래일 종가와 그때의 기준 고점으로 계산한 구간을 `복귀 기준 구간`으로 저장한다.
- 상태 종료 시점에 확정되어 있던 기준 고점을 `복귀 기준 고점`으로 저장한다.

## 전액 투자 종료

다음 중 먼저 충족한 조건에 따라 전액 투자 단계를 끝내고 기본 규칙의 정상 상태로 복귀한다.

### 한 구간 추가 하락

- 복귀 기준 구간을 저장한 다음 거래일부터 종가 구간을 비교한다.
- 종가 구간 번호가 복귀 기준 구간보다 커지면 다음 거래일 주문 후보를 만든다.
- 다음 거래일 시가 구간도 복귀 기준 구간보다 클 때 현재 시가 구간의 정상 목표 비중으로 리밸런싱한다.
- 시가가 복귀 기준 구간 이하로 회복하면 주문을 취소하고 TQQQ 100%를 유지한다.

### 이전 고점 돌파

- TQQQ 조정 종가가 복귀 기준 고점을 초과하면 그날 장 마감 후 전액 투자 단계를 끝낸다.
- 이미 TQQQ 100%이므로 별도 주문은 만들지 않고, 다음 거래일부터 기본 규칙의 정상 상태로 판단한다.
- 복귀 기준 고점과 정확히 같은 종가는 돌파로 보지 않는다.

전액 투자 단계가 한 번 끝나면 가격이 회복되더라도 다시 전액 투자 단계로 돌아가지 않는다. 이후 새로운 공황 또는 대공황이 발생하고 종료될 때만 이 추가 규칙을 다시 시작한다.

전액 투자 단계 중 공황 또는 대공황이 새로 시작되면 전액 투자 단계를 즉시 끝내고 해당 시장 상태의 기본 목표 비중과 전환 규칙을 우선 적용한다.
